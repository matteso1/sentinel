Up and Running in Seconds

No complicated setup. No YAML configurations. Just run.

terminal 
# Start the server
go run ./cmd/sentinel-server --port 9092

# Produce messages
sentinel-cli produce -topic events -message "Hello, World!" -count 100
# ‚úì Produced 100 record(s) to events (offset: 0)

# Consume messages
sentinel-cli consume -topic events -from-beginning
# offset=0 | key=(null) | value=Hello, World!-0
# offset=1 | key=(null) | value=Hello, World!-1
# ...
